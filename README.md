# Testes Unitários - JUnit, AssertJ e Mockito
<p>
  Projeto de API utilizando Java e Spring com foco em testes unitários utilizando as ferramentas JUnit, AssertJ e Mockito.
</p>

## Descrição
<p>Neste projeto foram desenvolvidos caso de testes para as operações de CRUD de uma entidade de Produtos, com casos de sucesso e casos de erro, de modo que todos os cenários fossem cobertos.</p>
